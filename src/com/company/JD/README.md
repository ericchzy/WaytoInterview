## 相似字符串

两字符串X与Y，如果存在a-z的某个排列组合，把X的第一个字符全换成a，第二个字符全换成b，就能把X变成Y，就表明两字符串相似。 
例如 xax与 aba，把z替换成a，a换成b就一样了，表明相似。 

求一个目标字符串X，用来匹配的字符串Y，求X的子串里有多少和Y相似的，输出相似字符串的个数。 
输入是X和Y，输出个数 

输入：
```$xslt
ababcdcbbab
xyx
1
2
```

输出：
```$xslt
4
```
